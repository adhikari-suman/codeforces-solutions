## 🧪 Environment Setup (using uv)

This project uses **uv** for fast and reliable Python environment management.

### 🚀 Create virtual environment (PyPy)

```bash
uv venv --python pypy3.10
```

### 📦 Install / Sync dependencies

```bash
uv sync --python pypy3.10
```

### ▶️ Activate environment

**Mac/Linux**
```bash
source .venv/bin/activate
```

**Windows**
```bash
.venv\Scripts\activate
```

### ✅ Verify setup

```bash
python --version
```

Expected output:
```
PyPy 3.10 (7.3.15, 64-bit)
```
